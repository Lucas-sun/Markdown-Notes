# 流媒体
### 协议
1. HLS
   - 由三部分组成: HTTP, M3U8, TS
     - HTTP是传输协议
     - M3U8是索引文件
     - TS是音视频的媒体信息
   - M3U8索引详解
```
#EXTM3U
#EXT-X-VERSION:3
#EXT-X-MEDIA-SEQUENCE:64
#EXT-X-TARGETDURATION:12 所有切片的最大时长
#EXTINF:11.550 ts切片的实际时长
livestream-64.ts
#EXTINF:5.250
livestream-65.ts
#EXTINF:7.700
livestream-66.ts
#EXTINF:6.850
livestream-67.ts
```
